    What should I know before starting training

    What should I do before the start of the course?

    While you are looking forward to the start of the course, it’s highly recommended taking the course Computer Scenes. This course helps you cover gaps (if any) in programming principles, organize your knowledge, and learn basic things about JavaScript. Thus, our education program would be easier for you. In case you have any questions do not hesitate to ask them on the appropriate discord channel or answer question from other students

    What base knowledge should I have before starting training

    • You should know basic data structures and their organization (array, list, stack, queue, tree, hash table, etc.).
    • You should be able to implement a simple algorithm in javascript language like basic algorithms for sorting and searching.
    • You should know the basic principles of OOP and be able to say a few words about each of them.
    • English language level: Intermediate (B1) and up.

    What might I do to improve my level of technical knowledge?

    • Read a comprehensive Javascript tutorial.
    • Use the Codewars platform to solve practical tasks. You can start with simpler ones.
    • Take free online courses:
      • Learn the Command Line
      • Learn Git
      • Algorithms
    • Believe in your strength!
